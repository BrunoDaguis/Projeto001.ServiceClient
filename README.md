# Projeto Exemplo .Net Core
###### Projeto exemplo para uso e estudo de ferramentas azure, CI/CD e etc

Nesse projeto foi utilizado:
- Asp.Net Core 3.1
- Injeção de Dependencia
- Repository Partner
- Banco SQL
- Entity Framework
- Autenticação JWT
- Azure Key Vault
- GitHub Actions
- Sonar Cloud
- Testes (Unitario e Integrado)
- AutoFixture e NSubstitute
- FluentAssertions
- Swagger
